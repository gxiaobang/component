## react组件库

### 项目说明
+ 基于react组件化封装
+ 用webpack进行项目打包
+ 项目热更新，实时反馈到浏览器

### 项目结构
* src 源目录
	* lib 工具类
	* components 基础组件
	* images 图片
	* fonts 字体图标
	* styles 样式
	* includes 页面段
	* pages 页面
	* stores 数据存贮
	
* dist 上线打包编译

* i18n 国际化


### 开发环境执行下面命令
```bash
$npm run dev
```

### 生产环境打包执行
```bash
$npm run build
```

### 请求代理
```bash
$npm run server
```

### 组件内容
+ Tab选项卡
+ Modal弹窗
+ Message消息提示
+ Select下拉框
+ Checkout勾选框
+ Spin加载中
