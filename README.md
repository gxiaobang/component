# react-component

## react组件库

### 项目说明
+ 基于react组件化封装
+ 用webpack进行项目打包
+ 项目热更新，实时反馈到浏览器

### 项目初始化安装
```bash
$npm run init
```

### 项目结构
* `assets`
	* `base` 基础工具类
	* `components` 视图组件
	* `containers` 业务组件
	* `styles` 样式
	* `routes` 路由

* `build` 用于发布上线打包

### 组件内容
+ Tab选项卡
+ Modal弹窗
+ Message消息提示
+ Select下拉框
+ Checkout勾选框
+ Spin加载中
