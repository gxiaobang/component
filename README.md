## react组件库

### 项目说明
+ 基于react组件化封装
+ 用webpack进行项目打包
+ 项目热更新，实时反馈到浏览器

### 项目初始化安装
```bash
$npm run init
```

### 项目结构
* assets 工作目录
	* utils 基础工具类
	* components 基础组件
	* containers 业务组件
	* impages 图片
	* fonts 字体图标
	* styles 公共样式
	* routes 静态页面路由
	

* build 用于发布上线打包

### 组件内容
+ Tab选项卡
+ Modal弹窗
+ Message消息提示
+ Select下拉框
+ Checkout勾选框
+ Spin加载中
